Selamat Datang

Website ini digunakan untuk implementasi CI/CD menggunakan GitHub Actions. Setiap perubahan akan langsung ter-deploy ke server tanpa proses manual.

Tentang

Fokus utama dari project ini adalah memahami bagaimana proses deployment modern bekerja. Mulai dari perubahan kode hingga website dapat diperbarui secara otomatis.

Project

CI/CD Pipeline

Pipeline dibuat menggunakan GitHub Actions yang mengirim file ke server melalui SSH dan rsync secara otomatis.

Auto Deployment

Setiap push ke repository akan langsung memicu proses deployment tanpa perlu upload manual ke server.
